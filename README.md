## ◎How to start in Local Env

### ・Backend Server
`cd back`  
class that contains the main method: `com.example.tomato.Main`  
execute `http://localhost:15000/tomato/{***}`  

### ・Frontend DevServer
`cd front`  
`npm ci`  
`npm run start`  
open `http://localhost:3000`
